### 풀이
1. 입력받은 4개의 좌표를 받을 때마다, 확인 칸에 채움
2. 확인해야 하는 좌표의 크기가 100임으로 충분한 시간복잡도 고려 가능
3. 확인 칸을 전부 채운 이후, 해당 칸 조사하여 개수 확인
4. 확인한 개수 출력

### 핵심 코드

<details>
<summary>코드 보기</summary>

```java
for (int x = coor[0]; x < coor[2]; x++) {
    for (int y = coor[1]; y < coor[3]; y++) {
        chk[x][y] = 1;
    }
}
```
- 정확히는 x와 y의 개념이 반대로 되어있지만, 코드 상으로는 문제 X
- x위치는 0와 2 위치의 값에 저장되어있으므로 해당 차이만큼을, y위치도 마찬가지로 1로 채움
